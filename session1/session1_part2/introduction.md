# 🚀 Session 1: Part 2 - Mars Discovery with Scratch

Welcome to the second part of Session 1! Today, we transition from writing Python code to building a fully functional **Mars Rover Exploration Game** using **Scratch**.

## 🔴 The Mission: Explore Mars
Mars is a treacherous planet filled with craters, steep slopes, and dunes. NASA sends rovers to these locations to collect data and carry out scientific studies. Your task today is to build a game that simulates a rover mission!

### 🎯 Learning Objectives:
- Master the **Scratch Environment** (Sprites, Backdrops, Stage).
- Use **Event-driven programming** to control your rover.
- Implement **Conditional logic** (If-Then) for hazards and science targets.
- Create a **Scoring System** and a **Mission Timer**.
