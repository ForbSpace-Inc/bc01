Summary of Section 1: Basics of Coding
===============================================================

You’ve just completed launch training!  
Just like astronauts must master the basics before flying to Mars, you have now learned the six essential building blocks every coder needs.

Here’s what you’ve achieved:

1. **Your First Program – Hello World**  
   Analogy: The very first “Houston, we have liftoff!” message from Mission Control.  
   Concept: Using `print()` to make the computer display messages on screen.

2. **Variables – Memory Boxes**  
   Analogy: Labeled cargo pods on your spaceship that store fuel, astronaut names, or oxygen levels.  
   Concept: Storing and reusing information (`name = "Alex"`, `age = 16`, etc.). You can change the value anytime!

3. **Getting Information from the User**  
   Analogy: Receiving live radio messages or voice commands from ground control or fellow astronauts.  
   Concept: Using `input()` to make your program interactive and ask questions.

4. **Doing Math & Operations**  
   Analogy: Calculating rocket speed, fuel needed for the journey, or days until landing on Mars.  
   Concept: Using `+ - * / ** %` to perform calculations inside your code.

5. **Making Decisions – if / else**  
   Analogy: The Mars rover’s brain deciding: “If there’s a boulder ahead, turn left. Else, keep driving.”  
   Concept: Using `if`, `elif`, and `else` so the program can react differently based on conditions.

6. **Repeating Things – Loops**  
   Analogy: A satellite orbiting Earth again and again, or the rover continuously scanning the same area for rocks.  
   Concept: Using `for` loops and `while` loops to repeat actions without writing the same code many times.

**In relation to space:**
- Print statements → Mission Control announcements  
- Variables → Onboard data storage (cargo pods)  
- Input → Commands from Earth or crew  
- Math → Navigation, fuel calculations, and trajectory planning  
- Conditionals (if/else) → Autonomous rover decisions  
- Loops → Continuous orbiting, scanning, and monitoring  

You have successfully completed the **Launch Pad** phase of coding!  
You are now ready to move into **Section 2: DEVELOPING FUN SPACE GAMES** and learn the **Four Pillars of Coding** (Variables, Conditionals, Loops, and Functions) to build real space adventures.
